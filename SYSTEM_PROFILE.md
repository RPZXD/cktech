# โปรไฟล์ระบบบริหารจัดการงานวิชาการ (CK Tech Academic System)

## ภาพรวมของระบบ (System Overview)
ระบบบริหารจัดการงานวิชาการ (Vichakan System v2.0) เป็นแพลตฟอร์มเว็บแอปพลิเคชันที่ออกแบบมาเพื่อเพิ่มประสิทธิภาพในการบริหารจัดการงานวิชาการภายในสถานศึกษา ช่วยให้การจัดเก็บข้อมูล การรายงานผล และการติดตามตรวจสอบเป็นไปอย่างเป็นระบบและสะดวกรวดเร็ว

---

## คุณสมบัติหลัก (Key Features)

### 1. ระบบสำหรับผู้บริหาร (Executive Dashboard & Admin)
*   **สรุปสถิติจำนวนรวม**: แสดงภาพรวมของรายงานทั้งหมด, จำนวนครูผู้สอน, จำนวนการนิเทศการสอน และจำนวนเกียรติบัตรที่มีในระบบ
*   **การจัดการข้อมูลพื้นฐาน**: จัดการข้อมูลครู, รายวิชา, แผนกวิชา และปีการศึกษา/ภาคเรียน
*   **ระบบตรวจสอบและอนุมัติ**: ตรวจสอบและอนุมัติเกียรติบัตรหรือรายงานต่างๆ ที่ครูส่งเข้ามา
*   **การตั้งค่าระบบ**: สามารถปรับเปลี่ยนข้อมูลโรงเรียน และการตั้งค่าพื้นฐานของระบบได้

### 2. ระบบสำหรับครูผู้สอน (Teacher Module)
*   **บันทึกการสอน (Teaching Report)**: บันทึกรายละเอียดการเรียนการสอนในแต่ละคาบ พร้อมระบบแนบหลักฐาน
*   **การเช็คชื่อเข้าเรียน (Attendance)**: บันทึกการเข้าเรียนของนักเรียนในแต่ละรายวิชา
*   **ตารางสอนและตารางเวลา (Timetable & Calendar)**: แสดงตารางสอนส่วนตัวและปฏิทินกิจกรรมทางวิชาการ
*   **การนิเทศการสอน (Supervision)**: บันทึกและติดตามผลการนิเทศการสอน
*   **การจัดเก็บเกียรติบัตร (Certificate)**: อัปโหลดและจัดเก็บข้อมูลการเข้าอบรมหรือเกียรติบัตรทางวิชาการ
*   **การเชื่อมโยงข้อมูลนักเรียน (Student Link & Analysis)**: วิเคราะห์ข้อมูลนักเรียนเบื้องต้นเพื่อใช้ในการจัดการเรียนการสอน

### 3. ระบบวิเคราะห์และรายงาน (Stats & Reporting)
*   **รายงานสถิติการสอน**: สรุปจำนวนชั่วโมงสอนและภาระงานครู
*   **รายงานสถิติการเช็คชื่อ**: วิเคราะห์อัตราการเข้าเรียนของนักเรียน
*   **รายงานการนิเทศ**: สรุปผลการนิเทศเพื่อนำไปปรับปรุงคุณภาพการจัดการเรียนการสอน
*   **ระบบพิมพ์รายงาน**: สามารถสั่งพิมพ์รายงาน (PDF) สำหรับเก็บเป็นหลักฐานการปฏิบัติงาน (เช่น `print_report.php`, `print_supervision.php`)

### 4. ระบบวิเคราะห์ข้อมูลนักเตรียน (Student Analysis)
*   **ข้อมูลรายบุคคล**: ดูข้อมูลพื้นฐานและผลการวิเคราะห์ข้อมูลนักเรียนเพื่อการดูแลช่วยเหลือนักเรียน (Student Support System)

## เทคโนโลยีที่ใช้ (Technology Stack)
*   **Frontend**: HTML5, Modern CSS (Glassmorphism design), JavaScript (ES6+), FontAwesome Icons
*   **Backend**: PHP (MVC Architecture)
*   **Database**: MySQL/MariaDB
*   **Pattern**: Model-View-Controller (MVC) เพื่อการจัดการโค้ดที่เป็นระบบและง่ายต่อการบำรุงรักษา

---

## จุดเด่นของระบบ (Highlights)
*   **User Experience (UX)**: ส่วนติดต่อผู้ใช้มีความทันสมัย (Modern UI) สวยงาม และใช้งานง่าย รองรับการแสดงผลทุกหน้าจอ
*   **Data Integrity**: มีระบบตรวจสอบสิทธิ์การเข้าใช้งาน (Auth Check) แบ่งระดับผู้ใช้งานชัดเจน (Admin, Teacher, Staff, Student)
*   **Comprehensive Analytics**: มีระบบสถิติที่ช่วยให้ผู้บริหารมองเห็นภาพรวมการดำเนินงานได้ในทันที
*   **Digital Workflow**: เปลี่ยนกระบวนการทำเอกสารกระดาษให้เป็นระบบดิจิทัล (Paperless) ลดภาระงานเอกสารของครูผู้สอน
